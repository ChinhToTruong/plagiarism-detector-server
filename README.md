# [PLAGIARISM DETECTOR SERVER](https://github.com/ChinhToTruong/plagiarism-detector-server)
***
## About
***
* This project will detect plagiarism in your document

## Requirements
***
* Java 17
* JDK 21
* Maven 3

## Technical
***
* Spring Boot 3
* Spring Security 6
* Redis
* MySQL
* Kafka
* Docker
* ELK
* Grafana, Prometheus
* Unit test

## Package application
***
- Clean application
```bash
    mvn spring-boot:clean
```
- Package application
```bash
    mvn spring-boot:package
```
## Run application
***
- Run
```bash
   mvn spring-boot:run   
```
- Docker
```bash
    docker-compose up 
```
